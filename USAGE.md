# TalkTranscript — ユーザーガイド

マイクとスピーカーの音声をリアルタイムに文字起こしし、テキストファイルに記録するツールです。  
通話・会議・打ち合わせの内容を自動で書き起こします。リアルタイム翻訳にも対応しています。

## はじめに

`TalkTranscript.exe` をダブルクリック、またはコマンドプロンプト/PowerShell から実行してください。

```
TalkTranscript.exe
```

初回起動時にマイクとスピーカーの選択画面が表示されます。
お使いの環境の GPU / CPU / メモリを自動検出し、最適な認識エンジンと GPU バックエンドを選択します。

> **音声認識モデルについて:**
> 初回実行時に認識エンジンのモデルファイルが自動でダウンロードされます。
> ダウンロード先: `%LOCALAPPDATA%\TalkTranscript\Models\`

---

## 認識エンジン

| エンジン | 特徴 | モデルサイズ |
|---------|------|------------|
| **Vosk** | 軽量・リアルタイム認識 | ~48MB |
| **Whisper tiny** | 高速・準リアルタイム | ~39MB |
| **Whisper base** | バランス型 (推奨) | ~142MB |
| **Whisper small** | 高精度 | ~466MB |
| **Whisper medium** | さらに高精度 (GPU 推奨) | ~1.5GB |
| **Whisper large** | 最高精度 (GPU 推奨) | ~3.1GB |
| **SAPI** | Windows 標準 (追加ダウンロード不要) | — |

エンジンは設定メニューまたはコマンドライン引数で変更できます。

### Whisper 後処理

Vosk や SAPI で録音した音声を、通話終了後に Whisper で再認識して精度を向上させることもできます。
設定メニューまたはコマンドラインから有効化できます。

## 録音中の操作

| キー | 操作 |
|------|------|
| **Ctrl+Q** | 録音を停止して終了 |
| **F2** | 設定メニューを開く |
| **Ctrl+B** | ブックマークを挿入 (重要箇所のマーク) |
| **Ctrl+L** | 言語切替 — 翻訳有効時: 翻訳の言語ペアを入れ替え (例: ja→en ⇔ en→ja)。翻訳無効時: 認識言語を en ⇔ 直前の非英語言語で切り替え |
| **↑ / ↓** | 表示を 1 行スクロール |
| **PgUp / PgDn** | 表示を 10 行スクロール |
| **Home** | 最初のエントリへ移動 |
| **End** | 最新のエントリへ移動 |

録音中は画面にリアルタイムで文字起こし結果と音量メーター (dB 対数スケール) が表示されます。
翻訳が有効な場合は、各発言の下に `↳ 翻訳テキスト` が表示されます。

## 設定メニュー

**F2** キー (録音中) または起動時に `--config` オプションで設定メニューを開けます。

```
TalkTranscript.exe --config
```

| # | 項目 | 説明 |
|---|------|------|
| 1 | エンジン | 認識エンジンの切り替え |
| 2 | GPU バックエンド | CUDA / Vulkan / 自動 / 無効 の選択 |
| 3 | 言語 | 認識言語 (日本語/英語/中国語/韓国語/フランス語/ドイツ語/スペイン語/自動) |
| 4 | 翻訳 | リアルタイム翻訳の有効/無効・言語ペア・翻訳対象・GPU 使用 |
| 5 | 出力ディレクトリ | 文字起こしファイルの保存先 |
| 6 | 出力フォーマット | SRT / JSON / Markdown の追加出力 |
| 7 | デバイス | マイク/スピーカーの選択 |
| 8 | 録音保存 | 音声を WAV ファイルとして保存 (マイクのみ/両方) |
| 9 | リソース制御 | CPU スレッド数/プロセス優先度 |
| 10 | 設定をリセット | 全設定を初期値に戻す |

設定は `%APPDATA%\TalkTranscript\settings.json` に自動保存され、次回起動時に引き継がれます。

## 出力ファイル

文字起こし結果は `Transcripts/` フォルダ (または設定で指定したフォルダ) に保存されます。

```
Transcripts/
  transcript_20260228_143000.txt     # テキスト (常に出力)
  transcript_20260228_143000.srt     # SRT 字幕 (オプション)
  transcript_20260228_143000.json    # JSON (オプション)
  transcript_20260228_143000.md      # Markdown (オプション)
```

| フォーマット | 用途 |
|-------------|------|
| **テキスト (.txt)** | 汎用 (常に出力) |
| **SRT (.srt)** | 動画編集ソフトの字幕として使用可能 |
| **JSON (.json)** | プログラムからの再利用・分析向け |
| **Markdown (.md)** | ドキュメント共有向け |

### 録音の WAV 保存

設定メニューの「録音保存」を有効にすると、通話終了時に音声データを WAV ファイルとして保存できます。

- **マイク + スピーカー** (既定) — `*_mic.wav` と `*_speaker.wav` の 2 ファイルを保存
- **マイクのみ** — `*_mic.wav` のみ保存 (スピーカー録音を残さない)

「マイクのみ」モードは、自分の発言だけ録音を残したい場合に便利です。
この設定はファイル保存のみに影響し、文字起こしの動作には影響しません。

### 出力例

```
通話記録 - 2026/02/28 14:30
============================================================

[14:30:05] 自分: お疲れ様です。
[14:30:08] 相手: お疲れ様です。本日の議題ですが...

  ★ [14:35:12] ブックマーク

[14:35:15] 自分: 了解しました。

============================================================
通話時間: 00:10:23  |  自分: 15件  |  相手: 12件
```

## コマンドライン引数

| 引数 | 説明 | 例 |
|------|------|----|
| `--engine <名前>` | エンジンを指定 | `--engine whisper-base` |
| `--cpu` | GPU を使わず CPU で動作 | `--cpu` |
| `--cuda` | CUDA バックエンドを明示指定 | `--cuda` |
| `--vulkan` | Vulkan バックエンドを指定 | `--vulkan` |
| `--lang <コード>` | 認識言語を指定 (`ja`/`en`/`zh`/`ko`/`fr`/`de`/`es`/`auto`) | `--lang en` |
| `--format <フォーマット>` | 出力フォーマットをカンマ区切りで指定 | `--format srt,json,md` |
| `--config` | 設定メニューを開く | |
| `--whisper-only [サイズ]` | Whisper モデルの事前ダウンロード | `--whisper-only large` |
| `--diag` | 診断モード (環境チェック) | |
| `--test` | テストモード (短時間で自動停止) | |

### 使用例

```powershell
# デフォルト (日本語) で起動
TalkTranscript.exe

# Whisper base エンジンを GPU で使用
TalkTranscript.exe --engine whisper-base

# CPU モードで起動
TalkTranscript.exe --engine whisper-base --cpu

# Vulkan バックエンドで起動 (AMD / Intel GPU)
TalkTranscript.exe --engine whisper-base --vulkan

# 英語認識で起動
TalkTranscript.exe --lang en

# SRT と JSON を同時出力
TalkTranscript.exe --format srt,json

# Whisper large モデルを事前ダウンロード
TalkTranscript.exe --whisper-only large
```

## GPU アクセラレーション (任意)

GPU を使って Whisper の認識速度や翻訳の推論速度を大幅に向上できます。

### サポートする GPU バックエンド

| バックエンド | 対応 GPU | 必要なもの |
|------------|----------|-----------|
| **CUDA** | NVIDIA (GeForce / RTX) | [CUDA Toolkit 13](https://developer.nvidia.com/cuda-downloads) |
| **Vulkan** | NVIDIA / AMD / Intel | GPU ドライバー (追加インストール不要) |
| **DirectML** | NVIDIA / AMD / Intel | 翻訳の GPU 推論に使用 (ランタイム同梱) |
| **自動** | — | 環境に応じて最適なバックエンドを自動選択 |

### 選択ロジック

- **NVIDIA GPU** → CUDA を優先。利用不可なら Vulkan にフォールバック
- **AMD / Intel GPU** → Vulkan を自動選択
- **GPU なし** → CPU モードで動作 (特別な設定は不要)

## リアルタイム翻訳

MarianMT (Helsinki-NLP) ベースのローカル翻訳に対応しています。クラウド接続は不要です。

### 対応言語ペア

| 言語ペア | 方向 |
|---------|------|
| ja ↔ en | 日本語 ⇔ 英語 |
| en ↔ zh | 英語 ⇔ 中国語 |
| ko → en | 韓国語 → 英語 |
| en ↔ fr | 英語 ⇔ フランス語 |
| en ↔ de | 英語 ⇔ ドイツ語 |
| en ↔ es | 英語 ⇔ スペイン語 |

### 設定方法

設定メニュー (**F2** → 「翻訳」) から以下を設定できます:

- **有効/無効** — 翻訳機能のオン/オフ
- **言語ペア** — 翻訳元 → 翻訳先の言語
- **翻訳対象** — "相手" / "自分" / "両方" を選択
- **GPU 使用** — DirectML による GPU 推論の有効/無効

翻訳モデルは初回使用時に自動でダウンロードされます。
録音中に **Ctrl+L** で翻訳方向を素早く入れ替えできます。
翻訳が無効の場合は、**Ctrl+L** で認識言語を英語と直前の非英語言語の間で切り替えられます。

## トラブルシューティング

### マイクの音声が認識されない

1. `--diag` オプションで診断モードを実行:
   ```
   TalkTranscript.exe --diag
   ```
2. Windows の **設定 → プライバシー → マイク** でアプリのマイクアクセスが許可されていることを確認
3. マイクがミュートになっていないか確認

### 認識精度が低い

- より高精度なエンジンに変更 (例: Whisper base → small)
- マイクの音量が小さい場合は、Windows のサウンド設定でゲインを調整
- 雑音の多い環境ではヘッドセットマイクを推奨

### 動作が重い

- 設定メニューの「リソース制御」から CPU スレッド数を減らすか、プロセス優先度を下げる
- 軽量なエンジン (Vosk / Whisper tiny) に切り替え
- GPU が利用可能な場合は GPU モードを有効化

| データ | パス |
|--------|------|

| 認識モデル | `%LOCALAPPDATA%\TalkTranscript\Models\` |
| 翻訳モデル | `%LOCALAPPDATA%\TalkTranscript\Models\` |
